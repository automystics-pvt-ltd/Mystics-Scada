Build an enterprise-grade, multi-tenant Solar Monitoring, SCADA, Device Integration, Operations, and Maintenance SaaS Platform focused on rapid customer onboarding, universal device connectivity, and long-term scalability.

The primary objective is to create a platform where I can onboard any solar customer, regardless of the devices, gateways, SCADA systems, or monitoring systems they currently use. The platform must allow me to connect, configure, collect, monitor, analyze, and report data from any source without requiring software development for each new customer.

The platform must include a Universal Device & Data Integration Engine that supports direct device connectivity, third-party gateways, vendor cloud platforms, existing SCADA systems, utility systems, APIs, MQTT brokers, databases, FTP/SFTP servers, CSV/Excel imports, and custom data sources.

When onboarding a new customer, I should be able to:

- Create customer account.
- Create one or multiple solar plants.
- Configure devices, gateways, or external data sources.
- Map incoming data fields to platform parameters.
- Start automatic data collection.
- Generate dashboards, alarms, reports, and analytics automatically.
- Complete onboarding entirely through configuration without writing code.
